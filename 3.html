周三 20 十二月 2017 in Python

数字

1.整数与浮点数是按有没有小数点区分的

2.除法(/)永远返回一个浮点数。如要使用floor除法并且得到整数结果（丢掉任何小数部分），你可以使用 // (地板除)运算符；要计算余数可以使用 % 整数和浮点数的混合计算中，整数会被转换为浮点数:

    >>> 17 / 3  # classic division returns a float
    5.666666666666667
    >>>
    >>> 17 // 3  # floor division discards the fractional part
    5
    >>> 17 % 3  # the % operator returns the remainder of the division
    2
    >>> 5 * 3 + 2  # result * divisor + remainder
    17
    >>> 3 * 3.75 / 1.5
    7.5
    >>> 7.0 / 2
    3.5

3.** 运算符计算幂乘方

    >>> 5 ** 2  # 5 squared
    25
    >>> 2 ** 7  # 2 to the power of 7
    128

4.变量在使用前必须 “定义”(赋值)，否则会出错:

    >>> # try to access an undefined variable
    ... n
    Traceback (most recent call last):
      File "<stdin>", line 1, in <module>
    NameError: name 'n' is not defined

5.除了 int 和 float，Python 还支持其它数字类型，例如 Decimal 和 Fraction。Python 还内建支持 复数 ，使用后缀 j 或 J 表示虚数部分（例如，3+5j）。 复数什么的已经忘了(≖ᴗ≖๑)
