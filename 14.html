Python json模块

周四 18 一月 2018 in Python
#Python

一个Python数据结构转换为JSON

import json

data = {
    'name' : 'ACME',
    'shares' : 100,
    'price' : 542.23
}

json_str = json.dumps(data)

dumps参数
indent 缩进
sort_keys 输出时字典的是按键值排序的，而不是随机的
separators是提供分隔符，可以出去白空格，输出更紧凑，数据更小。默认的分隔符是(', ', ': ')，有空格的。
ensure_ascii默认true，写入中文是乱码时改false
res = json.dumps(dict,indent=4,ensure_ascii=False)
print(res)

JSON编码的字符串转换回一个Python数据结构

res_py = json.loads(res)
print(type(res_py))
print(res_py)

注意 json有dumps和dump两种方法
dump是将Python数据转成json写入文件
load是从文件读取转成json格式
json格式是str

with open('f1.txt','w',encoding = 'utf-8') as f:
