Python yaml 模块

周二 20 三月 2018 in Python
#Python

yaml中有以下基本类型：

字符串
整型
浮点型
布尔型
null
时间
日期

大小写敏感
使用缩进表示层级关系
缩进的空格数目不重要，只要相同层级的元素左对齐即可
同一层级的表示为字典
# 表示注释，从它开始到行尾都被忽略
- 表示列表

- name: fff
  age: 0
  job: Tester
- name: James
  age: 18

  输出如下：
  [{'job': 'Tester', 'age': 0, 'name': fff}, {'age': 30, 'name': 'James'}]
