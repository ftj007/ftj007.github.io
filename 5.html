Python 条件判断

周六 23 十二月 2017 in Python
#Python

1.if 语句 elif 可以0到任意多，else可以没有。

score = 80 #input('请输入你的分数：')
score = int(score)#input()返回的数据类型是str，str不能直接和整数比较，必须先把str转换成整数。
if score<60:
    print('不及格')
elif score >= 60 and score<80: #60<=score<=80
    print('及格')
elif score >=80 and score<90:
    print('良好')
else:
    print('优秀')

2.if语句的三元操作 如果true返回x,false返回y。

x, y = 4, 5
samll = x if x < y else y #三元操作
print(samll)

3.if判断条件还可以简写，比如写： 只要x是非零数值、非空字符串、非空list等，就判断为True，否则为False。

if x:
    print('True')
